Инструкция по сборке приложения.

1. Требования к окружению.
  
  1.1 Java
    Для сборки приложения требуется версия Java не ниже 1.6.
  
  1.2 Apache Maven
    Сборка приложения осуществляется с помощью Apache Maven.
    Требуется версия не ниже 3.0.
    
2. Структура проекта.
   core - модуль для взаимодействия с БД. Содержит уровни DAO и Service.
   dist - модуль сборки конечного дистрибутива.
   web - мудуль для реализации web-интерфейса. Содержит уровень Controller.
   
3. Сборка приложения.
   Для сборки приложения и запуска Unit-тестов можно использовать стандартные команды Apache Maven
   mvn install
   
   Результатом успешного выполнения будет инсталляционный пакет
   contact-manager-[VERSION].zip 
   в директории trunk/src/dist/target/
   
4. Unit-тесты.
   Unit-тесты используются для тестирования модуля core.
   Для тестирования в качестве СУБД используется H2 Database Engine.
   Натсройки соединения БД находятся в файле: trunk/src/core/src/test/resources/jdbc.properties
   При тестировании БД создается в памяти, при этом инициализация выполняется автоматически.
   Таким образом, при каждой сборке проекта БД всегда будет чистой.
   
   Настройки логгера для тестов находятся в файле: /trunk/src/core/src/test/resources/log4j.properties
